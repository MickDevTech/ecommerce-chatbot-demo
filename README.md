# ecommerce-chatbot-demo
A simple react chatbot integrated with a LLM service in Hugging Face

## Proyecto ecommerce-chatbot-demo

Este proyecto es un chatbot integrado con un servicio LLM de Hugging Face. Este chatbot puede ser utilizado para asistir a los usuarios de un comercio electrónico.

### Requerimientos
Para ejecutar este proyecto, necesitarás tener Docker instalado en tu máquina.
Además, necesitarás tener una cuenta en Hugging Face para obtener una API key y buscar el modelo que deseas utilizar, este modelo debe tener disponible el uso de Inference API.

### Ejecución
Para ejecutar el proyecto, ejecuta el siguiente comando en la terminal:

```bash
docker compose up -d
```
